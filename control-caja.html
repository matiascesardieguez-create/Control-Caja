Configurar Firebase

Para que múltiples usuarios compartan los mismos datos en tiempo real, necesitas conectar tu proyecto de Firebase. Solo se hace una vez.

  1. Ve a console.firebase.google.com
  2. Crea un proyecto nuevo (gratis)
  3. Ve a Realtime Database → Crear base de datos
  4. Ve a Configuración del proyecto → Tus apps → Web
